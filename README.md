# 🎮 Video Game Dashboard & Recommendation System

An interactive Streamlit application for analyzing video game sales data and discovering recommended games based on game attributes and user-selected preferences.

## Live Demo

https://video-game-dashboard.streamlit.app/

## Features

* Interactive video game sales dashboard
* Explore games by genre, platform, publisher, and release year
* Visualize sales trends through charts and graphs
* Game recommendation feature based on game characteristics
* Dynamic filtering and data exploration
* Responsive Streamlit interface

## Technologies Used

* Python
* Pandas
* NumPy
* Streamlit
* Matplotlib
* Seaborn
* Scikit-Learn

## Dataset

The application uses a video game sales dataset containing:

* Game Name
* Genre
* Platform
* Publisher
* Release Year
* Regional Sales
* Global Sales

## Installation

```bash
git clone <https://github.com/Oppurtunistic07/Video-game-sales-and-recommendation-dashboard.git>

pip install -r requirements.txt
streamlit run app.py or streamlit run app_test.py
```
